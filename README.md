# Lemon AI - Skin Disease Detection System

## Overview

Lemon AI is an AI-powered skin disease detection system that uses Deep Learning and Computer Vision techniques to identify skin conditions from images.

The system uses Convolutional Neural Networks (CNN), YOLO object detection, Transfer Learning, OpenCV, and Grad-CAM visualization for accurate disease classification and analysis.

## Features

- Skin disease image classification
- CNN-based disease prediction
- YOLO object detection
- Transfer learning implementation
- Image preprocessing and augmentation
- Grad-CAM visualization
- Real-time webcam detection
- Dataset cleaning and preparation
